AquaMonitor:
O Controle Total da Qualidade Hídrica

Plataforma desenvolvida por **Tarcio** - Java/Spring Boot Developer

AquaMonitor oferece monitoramento em tempo real de parâmetros como **pH, Turbidez e Oxigênio Dissolvido**. Solidez, precisão e um design intuitivo para a gestão dos recursos hídricos.

Conheça a Tecnologia

Recursos Essenciais para o Monitoramento

Monitoramento Multi-Parâmetro

Acompanhe pH, Temperatura, Turbidez, Condutividade e Oxigênio Dissolvido simultaneamente e em tempo real.

Sistema de Alertas Inteligentes

Receba notificações imediatas (Críticas, Elevadas, Informação) quando os parâmetros saírem da faixa normalizada.

Back-end Java/Spring Boot

Construído por **Tarcio** com a performance e confiabilidade do Java para processar grandes volumes de dados de sensores.

Localização de Sensores

Integração com mapas para visualizar a distribuição dos dispositivos de coleta e o status de cada ponto monitorado.

A Arquitetura Desenvolvida por Tarcio

O coração do AquaMonitor é sua robusta API RESTful, desenvolvida em **Java 17+** e **Spring Boot**, garantindo escalabilidade e segurança. 

@Service
public class DadosService {
    // Criado por Tarcio - Java Spring Boot Developer
    public void processarLeituraSensor(Leitura leitura) {
        if (leitura.getTurbidez() > 40) {
            alertaRepository.salvarAlerta("Turbidez Crítica");
        }
        sensorRepository.atualizarLeitura(leitura);
    }
}
